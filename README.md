# Analog IC Design 

This repository documents my 5-day hands-on training in Analog IC Design
conducted at NI College, Mysore.

The training focused on understanding industry-oriented analog design concepts,
mixed-signal IC design flow, circuit analysis, and simulations using Cadence Virtuoso.

## Topics Covered

### Analog IC Design
- Basics of Analog IC Design
- Bandgap Reference (BGR) circuits
- Working principle of BGR
- Design methodology and calculations
- Temperature-independent reference generation

### Mixed-Signal IC Design Flow
- Introduction to mixed-signal IC systems
- Interaction between analog and digital blocks
- Complete design flow from specification to simulation
- Importance of verification and performance analysis
- 
## Analysis Performed
- DC Analysis  
  - Operating point verification
  - Biasing and region of operation analysis

- AC Analysis  
  - Gain and frequency response analysis
  - Bandwidth estimation

- Transient Analysis  
  - Time-domain behavior of circuits
  - Signal response verification

- Parametric Analysis  
  - Parameter variation and performance evaluation
  - Sensitivity analysis for design robustness

## Simulations
- All simulations were performed using Cadence Virtuoso
- Circuit behavior was verified using waveform and plot analysis
- Emphasis on understanding stability, accuracy, and performance

## Repository Contents
- Theory notes on Analog and Mixed-Signal IC Design
- BGR circuit explanation and analysis
- Design calculations
- Simulation result screenshots and observations

## Key Learning Outcomes
- Practical understanding of Analog and Mixed-Signal IC Design
- Hands-on experience with Cadence Virtuoso
- Ability to analyze circuits using DC, AC, transient, and parametric methods
- Improved understanding of real-world IC design workflows
